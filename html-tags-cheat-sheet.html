Headings

Headings from 'h1' to 'h6' identify blocks and sections. Every page must have one 'h1'.

Paragraphs

Paragraphs of text go in 'p' tags.

Lists

Unonrdered List

Unordered list are marked with bullet points an go in 'ul' tags. Each list item goes in an 'li' tag.

Types of lists:

  • Unordered lists (`ul`)
  • Ordered lists (`ol`)

Ordered Lists

Ordered lists are numbered and go in `ol` tags. Each list item goes in a `li` tag.

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the `href` attibute. Go to DuckDuckGo to search for more information.

Setting `target="_blank"` will open the link in a new window. See MDN for more details.

Text Emphasis

Italic

The `em` element adds extra emphasis to the test within the openning and closing tags. This shows up as italic text in most browsers.

Bold

You can also use the `strong` element to emphasize text. This shows up as bold in most browsers.
